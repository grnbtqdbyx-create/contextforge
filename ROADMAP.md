# Roadmap

## Milestone 0: Public Launch Foundation

- Apache-2.0 license, NOTICE, trademark policy
- README, roadmap, contribution guide, security policy
- TypeScript CLI skeleton
- CI for build, typecheck, and tests

## Milestone 1: Session Scanners

- Claude Code JSONL scanner
- Codex JSONL scanner
- fixture-backed demo mode
- graceful local fallbacks

## Milestone 2: Token Usage Analytics

- provider, project, kind, and session summaries
- cached token parsing where available
- expensive record list

## Milestone 3: Context and Cache Audits

- `AGENTS.md`, `CLAUDE.md`, `.cursorrules`, `.clinerules` health checks
- cache stability score
- volatile prefix detection

## Milestone 4: Context Pack Generator

- task-specific file ranking
- token budget control
- secret-aware Markdown export

## Milestone 5: Self-Learning Improvements

- suggestions from successful and failed sessions
- patch mode
- optional PR mode

## Milestone 6: v0.1.0 Release

- npm package metadata
- demo report screenshot or GIF
- public launch post

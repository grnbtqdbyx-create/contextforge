# Contributing

ContextForge is open source and welcomes contributions.

## Developer Certificate of Origin

This project uses DCO. By contributing, you certify that you have the right to
submit your contribution under the Apache License 2.0.

Use signed-off commits:

```bash
git commit -s -m "Add Claude scanner fixture"
```

## Local Development

```bash
pnpm install
pnpm test
pnpm build
pnpm contextforge scan --demo
```

## Good First Issues

Good starter contributions include:

- new fixture formats
- parser edge cases
- README examples
- context audit heuristics
- report styling improvements

## Future CLA Note

If the project grows and needs commercial relicensing or enterprise packaging,
ContextForge may introduce a Contributor License Agreement. Existing
contributions remain governed by the license under which they were submitted.
